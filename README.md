# Finance & Payments Data Platform (Azure Data Factory)

This project simulates a real-world finance and payments data platform built using Azure Data Factory.

## Objectives
- Ingest high-volume payment transaction data
- Integrate external financial APIs (FX rates, gateway status)
- Handle real-world failures and late-arriving data
- Build Bronze, Silver, and Gold data layers
- Implement monitoring, logging, and cost optimization

## Tech Stack
- Azure Data Factory
- Azure Data Lake Gen2
- Azure SQL Database
- Azure Key Vault
- GitHub (version control & documentation)
